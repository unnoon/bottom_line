bottom_line

- Adds face to your code

Javascript utility functions that work on any Collection (Iterable(Iterators)|Objects) or Sequence (any[]|string). Harnessing the power of generators & decorators and coming in handy per function packages.

Installation

npm install bottom_line

Features

  • Fully tested
  • Fully documented
  • Per function packages
  • Generator based
  • Works on any Collection (Iterable(Iterators)|Objects) or Sequence (any[]|string)
  • Decorators (that work on instance properties) & easy decorator creation tools
  • ES6 exports
  • Made with bits of love!
